# DOL-Compatibility-Mods

一些用於修正 **Degrees of Lewdity** 本體更新後，導致第三方模組不相容或發生錯誤的兼容補丁。

## 目前包含

### Head Mask Compatibility（頭部遮罩相容）

修正新版(`0.5.10.12+`) Head Mask 系統導致部分舊版頭部服裝模組顯示異常的問題。

### Clothing Trait Compatibility（服裝特質相容）

自動補齊第三方服裝模組使用用但遊戲不存在的服裝特質，避免服裝商店等場景因缺少特質而發生錯誤。

## 安裝方式

各模組皆為獨立模組，可依需求單獨安裝。

## 注意事項

- 請依照各模組說明確認適用的遊戲版本。
- 本倉庫中的模組僅提供相容性修正。
